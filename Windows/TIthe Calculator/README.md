# Tithe Calculator (Tkinter GUI)

A simple Python desktop application built with Tkinter to calculate tithe amounts based on income and a specified percentage. Designed with a clean interface and beginner-friendly code structure. 


# Features

* Easy-to-use graphical interface.

* Calculates tithe based on income and percentage.

* Default tithe percentage set to 10%.
