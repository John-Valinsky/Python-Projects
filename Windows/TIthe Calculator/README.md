# Tithe Calculator (Tkinter GUI)

A simple Python desktop application built with Tkinter to calculate tithe amounts based on income and a specified percentage. Designed with a clean interface and beginner-friendly code structure. 


# Features

* Easy-to-use graphical interface.

* Calculates tithe based on income and percentage.

* Default tithe percentage set to 10%.

* Displays result formatted in Indian Rupees (â‚¹).

* Input validation with error alerts.


# Preview

Inputs

	* Income

	* Tithe Percentage

Output

	* Calculated Tithe Amount


# Requirements

* Python 3.x.

* Tkinter (included with standard Python installations).


# How It Works

* Enter your income.

* Enter the tithe percentage (default is 10%).

* Click Calculate.

* The app displays the tithe amount instantly.


# Code Highlights

* Uses tkinter for GUI components.

* Exception handling prevents invalid input crashes.

* Clean separation between UI and calculation logic.
